# italian_news_analysis

## 개요
>"K-드라마"에 관한 이탈리아 온라인 뉴스 키워드 빈도 분석입니다.
- **📰 이탈리아 온라인 뉴스 언론사**
  - [la Repubblica](https://www.repubblica.it/), [ANSA](https://www.ansa.it/)
- **🔍 검색 키워드**
  - *serie sudcoreana*, *serie coreana*, *K-Drama*
- **📅 기사 수집 기간**
  - 2021.01 ~ 2022.04
- **🔡 이탈리아어 자연어 처리 패키지**
  - [spaCy](https://spacy.io/), [NLTK](https://www.nltk.org/)
